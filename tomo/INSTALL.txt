Dependencies:

* Django, version 1.3 or later, https://www.djangoproject.com/download/
* MarkDown, http://www.freewisdom.org/projects/python-markdown/

Installation:

1. Install dependencies.

2. Download Tomo application and put it on your Python path.

3. Add 'tomo' to your project's settings.INSTALLED_APPS.
   Since 'tomo' currently uses the Django admin application in some places,
   be sure to add 'django.contrib.admin' to settings.INSTALLED_APPS, if it is
   not there already.

4. Add 'django.core.context_processors.request' to your project's
   settings.TEMPLATE_CONTEXT_PROCESSORS. If your settings do not contain
   TEMPLATE_CONTEXT_PROCESSORS, use the following:

    TEMPLATE_CONTEXT_PROCESSORS = (
        # These are the default context processors provided by Django 1.3
        'django.contrib.auth.context_processors.auth',
        'django.core.context_processors.static',
        'django.core.context_processors.debug',
        'django.core.context_processors.i18n',
        'django.core.context_processors.media',
        'django.contrib.messages.context_processors.messages',
        # This is the context processor we want to include
        'django.core.context_processors.request',
    )

5. Run the following commands:

    cd /path/to/your/project/
    ./manage.py syncdb
    ./manage.py runserver
